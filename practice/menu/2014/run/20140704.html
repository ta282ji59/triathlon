説明
本日の練習は、七ヶ浜の選手とそうでない人で練習内容を分けます。
まず選手全員200mインターバルを行います。
七ヶ浜の選手は刺激入れ、それ以外の人はレペティション前の慣らしを
目的とした練習ですので、どちらの選手も流すように走って下さい。
その後、七ヶ浜の選手は30分間jogを行って下さい。
それ以外の選手は、レペティションを行います。
レペティションとは、トレーニング形態の一つで、
運動と疲労が完全に回復するまでの休息を繰り返す方式のトレーニングです。
ランニング、筋力トレーニング、技術練習などを織り交ぜ、有酸素性持久力
無酸素性持久力、スピード持久力、筋力などを高めることを主目的とします。
100% の力で走り、最後の選手が戻ってから10分完全休養してください。
