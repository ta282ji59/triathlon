説明
本日の練習は、長井トラの選手とそれ以外の選手でメニューを分けます。
長井トラの選手は、200mインターバルで刺激入れをした後
30分jogで疲労抜きを行ってください。
それ以外の選手は、内周を使ってビルドアップを行います。
ビルドアップ走は、走るスピードを徐々に上げていくトレーニングです。
ビルドアップ（BUILD UP）とはカラダを鍛え上げるという意味を持ち、
長距離をしっかり走りきれる強いカラダをつくりあげる練習メニューといえます。
体に負荷をかけるトレーニングなので
練習前後のボディケアは入念に行うようにしてください
内周2周につき10秒ずつタイムを上げてください。
