説明
本日の練習は内周を使ってビルドアップ走を行います。
ビルドアップ走は、走るスピードを徐々に
上げていくトレーニングです。
そもそもビルドアップ（BUILD UP）とは
「カラダを鍛え上げる」という意味を持ち
まさに長距離をしっかり走りきれる
強いカラダをつくりあげる練習メニューといえます。
体に負荷をかけるトレーニングなので
練習前後のボディケアは入念に行うようにしてください。
2kmごとに10秒タイムをあげるようにしてください。
今回の練習だと、5周ごとに1周のターゲットタイムを
2秒あげて、5周で合計10秒あがるように走って下さい。
