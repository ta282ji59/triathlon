説明
本日の練習はレペティションを行います。
レペティションとは、トレーニング形態の一つで、
運動と、疲労が完全に回復するまでの休息を繰り返す方式のトレーニングです。
ランニング、筋力トレーニング、技術練習などを織り交ぜ、有酸素性持久力
無酸素性持久力、スピード持久力、筋力などを高めることを主目的とします。
100% の力で走り、restは10分完全休養してください。
