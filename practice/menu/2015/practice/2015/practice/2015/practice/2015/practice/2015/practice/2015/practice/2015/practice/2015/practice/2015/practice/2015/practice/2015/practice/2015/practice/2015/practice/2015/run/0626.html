説明
　今日はミーティング終了後に大会前ということで、おしん選手は刺激入れを行います。
刺激入れとは大会の数日前に、全力の80〜90%の力で走ってレースペースより速いペースで走ることで脚筋を刺激し、超回復で強くなった脚筋がラクにレースペースに対応できるというものです。
超回復とは「筋肉を成長させるために必要な必要最小限の刺 激を与えていくこと」です。
ですので、終了後のjogではあまり速く走らないようにしましょう。
　それ以外の部員は自分のフォームを気にしながらjogを行ってください。
