説明
　　本日の練習は、主に大田原マラソンでの完走を目標としている人向けに、
60分時間走を行います。大田原マラソンでの打ち切りのタイムは
以下の通りとなっています。

《10kmの部》
＞3km地点…10時58分／スタート後18分
＞全コース60分をもって打ち切りとします。

　そのため、完走を目標としている人は自分が60分間走り続け、
どのくらいの距離を走ることができるか走ってみてください。
また、上記の3km地点での打ち切りは、大学の内周での距離に直すと、
3周(3.75km)で22分30秒がボーダーです。
そのあたりも考えて走ってみてください。

　それ以外の部員は60分間jogや、10kmペース走など、各自の練習を行ってください。
