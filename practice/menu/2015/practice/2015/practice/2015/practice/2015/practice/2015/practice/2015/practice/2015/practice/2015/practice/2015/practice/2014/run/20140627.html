説明
本日の練習はペース走と400mインターバルを行います。
内周を使ってペース走を行い、ペース走の後グラウンドで
400mインターバルを行います。
内周でペース走を行うのは、レースはコンクリートの上を走るため
グラウンドとは走っている感覚が少し変わってきます。
レースの感覚に慣れてもらうという意味合いで今回は内周で行います。
