03/10 (月) ラン練





準備体操
main: 90分LSD

説明
先日の松代、おつかれさまでした。
普段使わない筋肉を使ったので、筋肉痛の方もいるのかな？

本日の練習は内周90分LSDを行います。
LSDとはlong slow distanceの略で、
文字通り長い距離をゆっくりとしたペースで走る練習です。
ランでの基盤となる筋肉を鍛える練習です。
jogと違う点は、jogではある程度までペースをあげるので
心拍が上がり、リラックスした状態で走りますが、
LSDでは心拍は上がらせずに
走るようにしてください。
途中でペースをあげたくなるかと思いますが、
そこでペースをあげずに
自分のフォームを見直しながら走って下さい。
