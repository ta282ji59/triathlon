説明
本日の練習は、プライオメトリクス、アジリティー、反応ダッシュの後
うつくしまコースでjogを行います。

プライオメトリクス
筋肉を伸張させたあと、すぐに収縮させ、筋肉の爆発力を
アップさせることをめざすトレーニングです。
筋が腱を急激に引っ張ると、強い伸張反射、つまりパワーが生まれます。
引っ張る速度である筋力縮速度を向上させれば、より大きなパワーが期待できます。
ジャンプして、着地する時の足の面は小さく、次のジャンプにすぐ移ってください。

アジリティー
機敏,軽快さのトレーニングのことです。
静→動、動→静など動きの変化の速さを身に付けます。
俊敏性・機敏性など、スピードや瞬発力などの能力を向上させることを
目的としたトレーニングです。
マネージャーさんが出すサインの指示に従って集中して行ってください。

反応ダッシュ
機敏、軽快さのトレーニングです。
アジリティーと目的は同じですが、反応ダッシュのほうが
思考、判断速度が求められますので、集中して行ってください。
これは全員3回成功するまで続けます。
